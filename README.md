# ETL y Dashboard de Inteligencia de Datos: Reportes COVID-19

Este repositorio contiene el flujo de trabajo para el procesamiento masivo de datos y la visualización de indicadores epidemiológicos utilizando Python y Power BI.

## Descripción
El proyecto se centra en la integración de más de 1,000 archivos CSV con reportes diarios. Se realizó un proceso de limpieza y unificación de datos para generar una base de datos estructurada que permita el análisis de tendencias y la clasificación de riesgo por estado.

## Tecnologías
* Python: Pandas y OS para la automatización del proceso ETL.
* Power BI: Modelado de datos y diseño de dashboard interactivo.
* Google Colab: Entorno de desarrollo para el procesamiento inicial.

## Componentes del Proyecto
1. Automatización ETL: Script para la lectura, limpieza y consolidación de archivos dispersos.
2. Análisis Estadístico: Cálculo de medias móviles y tasas de crecimiento de contagios.
3. Dashboard de Inteligencia: Visualización de KPIs como el semáforo de riesgo y la saturación por periodos.
